# Selenium Test Automation Project

This project is a test automation framework for web applications using Selenium WebDriver, C#, NUnit, and Allure for reporting.

## Prerequisites

Before running the tests, ensure that the following tools are installed:

- [.NET SDK](https://dotnet.microsoft.com/download)
- [Selenium WebDriver](https://www.selenium.dev/documentation/webdriver/)
- [Allure](https://allure-framework.org/)
- [NUnit](https://nunit.org/)
